# Counting_Sheep
## 3D 술래잡기 게임

### 게임명
 Counting Sheep (가제)

### 장르
 3D 술래잡기

### 기술
 Unity 3D, C#, Github

### 설명
 각각의 다양한 맵중 하나를 선택하여 농부에게서 도망치며 아이템들을 모아 탈출하는 게임

### 특징
 1. 귀여운 동물 테마
 2. 난이도 조절을 위한 점점 늘어나는 농부
 3. 다양한 아이템 및 탈출 방법
